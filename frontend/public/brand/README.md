# KiSinWi — бренд-ассеты

Готовые файлы лого для статей, презентаций, README и т.п. Все файлы — векторные (SVG),
масштабируются без потери качества. Раздаются фронтендом по пути `/brand/...`.

Знак — глянцевый бейдж в стиле app-icon: залитый фирменным градиентом squircle с белой
монограммой «K», стеклянным бликом и кромкой.

## Файлы

| Файл | Что это | Когда использовать |
|------|---------|--------------------|
| `kisinwi-logo.svg` | Знак + надпись «KiSinWi», светлая надпись | На **тёмном** фоне |
| `kisinwi-logo-dark.svg` | Знак + надпись «KiSinWi», тёмная надпись | На **светлом** фоне |
| `kisinwi-mark.svg` | Только глянцевый знак (квадрат) | Иконка, аватар, маленькие места |
| `kisinwi-wordmark.svg` | Только надпись (градиент) | Когда знак не нужен |
| `../favicon.svg` | Знак для вкладки браузера | Favicon сайта |

Рядом с каждым SVG лежат PNG-версии (`*.png` — @1x и `*@2x.png` — @2x, прозрачный фон)
на случай, если нужен растр (например, площадки, не принимающие SVG).

> ⚠️ В PNG-версиях с надписью использован запасной жирный sans-serif: при экспорте на сборочной
> машине не был установлен фирменный шрифт Clash Display. Для точной типографики бери **SVG**
> (откроется правильным шрифтом в браузере) или переэкспортируй PNG, установив Clash Display.

## Цвета бренда

- Акцент (основной): `#b15e6b`
- Акцент (светлый): `#d08c96`
- Градиент знака: `#b15e6b → #d08c96` (135°)
- Текст на тёмном фоне: `#f0f4fa`
- Текст на светлом фоне: `#1d2330`

## Правила

- Сохраняй свободное поле вокруг лого не меньше высоты знака.
- Не растягивай и не перекрашивай знак произвольными цветами — только фирменный градиент.
- Для растровых нужд открой SVG в браузере и экспортируй в нужном размере (вектор резкий в любом масштабе).
