# Data Manipulation Service

### Памятка для разработчиков по созданию env

#### Разработка

1) Установите в своей системе uv по [инструкции](https://docs.astral.sh/uv/#installation)
2) Для установки виртуального окружения используйте команду:

```bache
uv sync
```

Если нужно создать виртуальную среду без библиотек испольуемых при разработке используйте при написании команды модификатор `--no--dev`
